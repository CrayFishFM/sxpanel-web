## Design Context

This project has captured design context for `/impeccable`:

- `PRODUCT.md` — register (brand), users, positioning, brand personality, anti-references
- `DESIGN.md` — the "Control Room" visual system: colors, typography, elevation, components, do's/don'ts
- `.impeccable/design.json` — machine-readable sidecar (tonal ramps, component snippets) for the live design panel

Read both before making visual changes to the marketing site or docs.
