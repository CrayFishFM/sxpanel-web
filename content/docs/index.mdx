**sxPanel** is a full overhaul built on top of [txAdmin](https://github.com/tabarra/txAdmin) —
designed as a **drop-in replacement** with full compatibility for existing txAdmin
servers, databases, and configurations.

<Callout title="Migrating from txAdmin?">
  Just replace the `monitor` folder in your FXServer artifacts with the sxPanel
  build. Existing `txData` directories work without modification.
</Callout>

## Why sxPanel?

sxPanel takes everything you know from txAdmin and rebuilds it into a faster,
more capable control room for your server — while staying 100% compatible with
your existing setup.

- **Drop-in replacement** — no data migration, no rewrites.
- **Self-contained** — player data lives in a local JSON database (no MySQL
  or other database server required).
- **Batteries included** — console, insights, player tools, Discord bot and an
  in-game admin menu, all in one build.
